## WorldEdit

### /we threads

```command
/we threads
```

現在の WorldEdit のスレッド情報を書き出す。

### /we version

```command
/we version
```

WorldEdit と FAWE のバージョン情報を表示する。

### /we help

```command
/we help [<コマンド>]
```

FAWE のコマンドヘルプを参照する。

### /we reload

```command
/we reload
```

設定を再読込する。

### /we changelog

```command
/we changelog
```

FAWE のバージョン履歴を確認する。

### /we cui

```command
/we cui
```

CUI ハンドシェイクを完了する（内部用途）。

### /we debugpaste

```command
/we debugpaste
```

hastebin.com にデバッグ情報を書き出してアップロードする。

### /we tz

```command
/we tz [タイムゾーン]
```

スナップショット用にタイムゾーンを設定する。
